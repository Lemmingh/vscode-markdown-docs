## Markdown style

| Element           | Style       |
| ----------------- | ----------- |
| Heading           | ATX heading |
| Bold              | `**`        |
| Italic            | `*`         |
| Unordered list    | `-`         |
| Fenced code block | ` ``` `     |
| Thematic break    | `---`       |

## Date and time

Times are always expressed in UTC (Coordinated Universal Time).

Please use ISO 8601 or a similar format as described below.

* `2013-04-05T06:07:09Z`
* `2013-04-05 06:07:09`
