## Markdown style

| Element           | Style       |
| ----------------- | ----------- |
| Heading           | ATX heading |
| Bold              | `**`        |
| Italic            | `*`         |
| Unordered list    | `-`         |
| Fenced code block | ` ``` `     |
| Thematic break    | `---`       |
