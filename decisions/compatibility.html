# Markdown preview

This extension contributes to VS Code's built-in Markdown preview.

It should work harmoniously with other extensions that follow VS Code's guide.

# Exporting to HTML

This extension uses markdown-it to render HTML, and tries to load Markdown extensions in the same way as VS Code.

It should produce nearly the same output as VS Code.
