# Math support

# Overview

LaTeX math rendering is powered by KaTeXopen in new window.

This extension currently only recognizes pandoc-style delimitersopen in new window:

  • $...$ for math span (inline).
  • $...$ for math block (display).

There are proposalsopen in new window for improving this. Any help is welcome.

For dedicated math support, now it's recommended to set this extension's markdown.extension.math.enabled to false, and use Markdown+Mathopen in new window.
